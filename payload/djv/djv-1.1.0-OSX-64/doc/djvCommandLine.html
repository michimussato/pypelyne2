Command Line Options

  • User Interface
  • OpenGL

    		•

    User Interface

    -reset_prefsDo not load the preference at start up.

    OpenGL

    -render_filter (zoom out) (zoom in)Set the render filter. Options = Nearest, Linear, Box, Triangle, Bell, B-Spline, Lanczos3, Cubic, Mitchell. Default = Linear, Nearest.
    -render_filter_highSet the render filter to high quality settings (Lanczos3, Mitchell).

    General

    -time_units (value)Set the time units. Options = Timecode, Frames. Default = Frames.
    -default_speed (value)Set the global speed. Options = 1, 3, 6, 12, 15, 16, 18, 23.976, 24, 25, 29.97, 30, 50, 59.94, 60, 120. Default = 24.
    -max_sequence_frames (value)Set the maximum number of frames a sequence can hold. Default = 172800.
    -debug_logPrint debug log messages.
    -help, -hShow the help message.
    -infoShow the information message.
    -aboutShow the about message.
