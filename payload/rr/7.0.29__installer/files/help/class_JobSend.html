Class _JobSend functions:

Name

Description

*

All functions of _JobBasics

bool

rrOption( int id)

returns the boolean value of rrOptions of the job. See "module rrJob"  for ids.

bool

clientAssigned ( int client_number)

returns true if the client is assigned for the job.


clientAssigned_Set( int client_number)

set the client assignment

int

clientValue_Get( int client_number, int valueID)

gets some information about the client regarding the job.

valueID

value

0

Number times the client got a segment of this job for rendering.

1

Number times the client was aborted.

2

Number of times the client crashed.

3

last segment the client got - status of the job

4

last segment the client got - thread ID of the client

5

last segment the client got - frame step

6

last segment the client got - frame start

7

last segment the client got - frame end

8

total frames rendered

9

last segment the client finished - time taken

10

total time the client spend on this job

int64

fileNumber2jobNumber(double filenumber, qint8 MultiFrameID)

A job counts frames internally 0,1,2,3,.... no matter which frame start or frame step the job has.

This function converts a frame number into the internal job number.

If you use tile frame, then you need to set MultiFrameID to the tile you want to access. Otherwise use MultiFrameID=0.

double

jobNumber2fileNumber(int64 jobNumber )

A job counts frames internally 0,1,2,3,.... no matter which frame start or frame step the job has.

This function converts the internal job number into a frame number.

int8

jobNumber2multiID(int64 jobNumber )

A job counts frames internally 0,1,2,3,.... no matter which frame start or frame step the job has.

If you use tile frame, this function converts the internal job number into the tile ID. Otherwise it always returns 0.

int

previewID2FrameNr(int id)

returns the frame number of preview frame with id. There are maximal previewNumberOfFrames.

QString

previewFilenameThumbnail(int id)

returns the file name of the preview - thumbnail version

QString

previewFilenameRGB(int id)

returns the file name of the preview - full size RGB version

QString

previewFilenameA(int id)

returns the file name of the preview - full size Alpha version
