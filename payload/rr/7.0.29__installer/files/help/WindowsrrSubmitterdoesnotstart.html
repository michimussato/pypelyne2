Windows: rrSubmitter does not start

Installation ›› Troubleshooting ››

If you cannot start the rrSubmitter from inside your 3D application, then check these steps:

  • Windows trusted network.
  • Open the file [RR]\win__rrSubmitter.bat with an editor/textpad.
    Check for the line
    set "RR_ROOT=......."
    This variable should be the path to your RR folder in the network.
