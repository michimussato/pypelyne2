Clients do not connect

Installation ›› Troubleshooting ››

Description of the problem:

The client is listed in rrControl.

But it is shown grey (not connected).

You have restarted the machine but it does still not help.




The first place you should look are the rrApplication Log Files.

To check if the client starts at all.

Is there any information in the log from the current date?


rrClient started manually

Checking the Application Log File should show you at least one error.

If not, try to start the application directly vai the RR/bin/... folder

rrClient running as service

in the background

If you have an entry in the Application Log File from the last time you restarted the machine/service,

then at least the client application startup was successfull.

This means that the issue is probably not the service.


If you have nothing in the Application Log File, then you should check the Service Log Files
