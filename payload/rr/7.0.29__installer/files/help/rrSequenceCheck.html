rrSequenceCheck

rrApplications ›› rrTools/Post-Script ››

rrSequenceCheck reads all images and checks for errors inside the images.

Then it displays the status of this check in the job table of rrControl or rrWebsite.


It also crops EXR files if the option is enabled for the job, see CropEXR.



If set in rrConfig, then it also moves the broken files that they are re-rendered.

See rrConfig/Image Verification
