  I do not see any clients in the list


You have all your scene data in the submitter (either via plugin or via loading the scene file).

But there are no or not all clients in the client table.


rrSubmitter shows the clients which have the render application of the active jobs installed.

If clients are not listed, then the render application is not set in their configuration.


You have two options:

  1. Open the configuration at the client.
    On the second tab you can either manually or automatically (via the button on the bottom) add the render application.

  2. If you have any clients, you probably do not want to login and add the application at every clients.
    So if the automated way with the button in 1. works, you can send that command to all clients.
    Open rrControl, select all clients and choose "Config"/"Rescan for installed software" as client command.
