Setenv Files

Customization ›› Render Applications ››


Location: [RR]\render_apps\_setenv


Setenv files are used to set the environment for the render application.

Some renderer or plugins require to set environment variables to their install location.

Setenv files are also used to tell the render application the location of the renderer preferences


Setenv files are called by the commandline <SetEnvSoft> in your render config.

RRender searches for the following files and executes the one it finds:

1. [RR]/render_apps/_setenv/[OS]/setenv_[renderapp]_[renderer]

2. [RR]/render_apps/_setenv/[OS]/setenv_[renderapp]


Environment variables set


The client sets a lot of environment variables before calling the setenv.

This way you have access to parameters like render application version.

So you can define different plugin locations for different versions of your software.


You can do this either by an if then switch or by directly using the variable in the path name.

See existing environment files for an example.



The client defines the following environment variables, you can check them by taking a look into render log files of a job.


rrExeBit                        The value is x32 or x64 depending on the render executable.

rrExeVersionMajor        The major version of the render application started.

rrJobVersionMinor        The minor version of the render application started.

rrJobRenderapp                The render application of this job (Softimage, Maya, Nuke...).

rrJobRenderer                The renderer used (mentalRay, VRay...)

rrJobRenderType        Same as RenderType in the render config, custom defined.

rrJobBit                        The bit restriction of the Job. x32, x64 or xAll (if not restricted at the submitter)

rrJobVersionMajor        Major version set for the job on submission.

rrJobVersionMinor        Minor version for the job

rrJobTiled                Is it a tiled render?

rrJobScene                Custom Scene name set at the submitter

rrJobShot                Custom Shot name set at the submitter

rrJobVersion                Custom Version name set at the submitter

rrClientName                Name of the client

rrClientCPUs                Total number of CPUs the client has.

rrClientCPUsUsed        Number of CPUs to use for render processes.

rrClientBit                x32 or x64 depending on the operation system of the client.

rrClientRenderInstance        The number of the render instance that was started. The value ranges from 0 to 10.

rrClientLocalRoot        Root of the Local Data Folder

rrLocalPrefs                The preference folder of the current render application in the Local Data Folder

rrClientLocalTemp        The temp folder in the Local Data Folder

TEMP                        Same as rrClientLocalTemp without slash at the end.

TMP                        Same as rrClientLocalTemp without slash at the end.

RR_ROOT                Root directory of the RR network folder.

rrBin                        Path to all executables in the RR network folder.

rrPlugins                Path to the plugins folder of the render application (usally [RR]/render_apps/renderer_plugins/[renderapp]/)

rrPrefs                        Path to the network preference folder of the render application ( [RR]/render_apps/renderer_prefs/[renderapp]/)

rrExe                        Path to the network executable folder of the render application ( [RR]/render_apps/renderer_exe/[renderapp]/)
