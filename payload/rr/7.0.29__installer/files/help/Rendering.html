Scene filename and client inside .EXR


If you render .exr and the client crops the EXR, then it also includes the name of the Softimage/Maya/.../ scene and the machine that it has rendered.

Those informations can be seen with an EXR viewer or with rrViewer.


Linear Render


If you render linear images, then enable "preview gamma 2.2" in the rrSubmitter.

All images shown in rrControl, rrWebsite and rrViewer will have gamma 2.2 applied to view the images right.
