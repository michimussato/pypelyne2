Enable RR to Shotgun Python module

RR has prepared a shotgun connection.

The first that it does is to prepare some functions that you can use any python script to connect to shotgun.

Then RR has some scripts included that can add render stats to shots in Shotgun.

The scripts are available via the rrSubmitter once you enable the Shotgun connection.
