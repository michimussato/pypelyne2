Render app licenses

Customization ›› Render Applications ››

A lot of renderer or plugins do not have unlimited render licenses.

Therefore you can tell RR how many render licenses you have.

See rrConfig, tab "Render Lic."


RRender is also able to reduce the number of licenses temporally.

If a client returns a job and a license error message was found in the log file, then the server reduces the licenses for this plugin for 1 hour.
