Location: [RR]\render_apps\renderer_plugins

RRender offers a shared location for renderer plugins for applications that support an external plugin path.

You do not have to use exactly these pre-defined directories, you can use any directory in your network if you like.


Some render applications can be informed about a plugin directory with either

an environment variable (e.g. MAYA_PLUGIN_PATH),

a commandline parameter (XSI -w WORKGROUP)

or config file (e.g. Lightwave)


These settings for the render application are defined in the Setenv Files.
