rrCopy

rrApplications ›› rrTools/Post-Script ››

rrCopy is a copy application that can be used to copy whole folders with subfolders.

It also supports a sync mode which does only copy changes files.


Please execute rrCopy --help for a list of commandline flags.


rrCopy location


You can either hardcode the path to rrCopy or use environment variables.

If you use variables, then the script will work in any RR environment.


Windows batch file:

%RR_ROOT%\bin\win\rrCopy


Linux shell script:

$RR_ROOT\bin\lx64\rrCopy


Render config commandline (All OS):

<rrBin>rrCopy



Examples



Sync a 3dsmax plugins from a source folder:

rrCopy -sync -d "\\Fileserver\Share\Installs\max\update10202\3ds Max 2015"   "c:\Program Files\Autodesk\3ds Max 2015"
