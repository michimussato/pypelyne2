Class _JobBasics functions:

Return type

Name

Description

bool

rrOption( int id)

returns the boolean value of rrOptions of the job. See "module rrJob"  for ids.

QString


custom_Str(const QString &name)

Get a custom data item of the job.

Each data item has a name and a value.

You can save up to 5000 letters. If unicode letters are found in value, then the string takes twice its length in bytes.

Reserved names are "UserInfo" and "EnvFile".

customSet_Str(const QString &name,const QString &value)

set custom data item. An empty value is not saved. If a data item with name exists, it is overwritten or deleted (if value is empty).
