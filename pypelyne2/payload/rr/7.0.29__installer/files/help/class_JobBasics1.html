Class _JobBasics functions:

Return type

Name

Description


int


waitForPreID(int index)

This is only used for jobs at submission.

Get the waitForPreID of this job. A job has up to 20 waitforIDs.

When the job is received by the server, the server checks for other jobs send from the machine in the last 15 minutes.

The server then gets the latest job with the same PreID as a WaitForPreID of this job, then this jobs waits for the other job.

Index ranges from 0 to maxWaitForPreID. Maximum is 20.

rrString

setWaitForPreID(int index, quint8 value )

Set the waitForPreID of this job. A job has up to 20 waitforIDs.

You have to update maxWaitForPreID as well.

QString


custom_Str(const QString &name)

Get a custom data item of the job.

Each data item has a name and a value.

You can save up to 5000 letters. If unicode letters are found in value, then the string takes twice its length in bytes.

Reserved names are "UserInfo" and "EnvFile".

customSet_Str(const QString &name,const QString &value)

set custom data item. An empty value is not saved. If a data item with name exists, it is overwritten or deleted (if value is empty).
