Daily Execute

Customization ››

It is possible to define batch scripts that are executed once a day by the client or server.


Depending on the application and the OS the batch scripts has to be named:

[RR]\sub\dayexecute_client_win.bat

[RR]\sub\dayexecute_client_lx.sh

[RR]\sub\dayexecute_client_mac.sh


[RR]\sub\dayexecute_server_win.bat

[RR]\sub\dayexecute_server_lx.sh

[RR]\sub\dayexecute_server_mac.sh
